افزونه پرداخت دلخواه وردپرس

== Description ==
this is an iranian payment plugin that can be used as a custom payment system or a donaite system , so users can get money from users easiy,one of this pluin is that it is modular and can be extended to support more gateways.

با این پلاگین میتونید سیستم پرداخت خودتون رو راه اندازی کنید.
تا اکنون این پلاگین بر روی درگاه های زیر توانایی کار کردن را داراست:
+ درگاه ملت
+ واسط زرین پال
+ واسط پی لاین (pay.ir)
+ ایران کیش
+ سداد /بانک ملی ( درگاه جدید)
+ مبنا کارت
+ مبنا کارت(جدید)

امکانات:

*    قابل استفاده در درگاه های مختلف (ماژولار)
*    ساخت فرم و افزدن فیلدهای سفارشی از مدیریت
*    تنظیم ارسال پیامک به هنگام پرداخت جدید
*   ثبت تمام پرداختی ها با عناوین موفق/ناموفق/در انتظار پرداخت
*   مجهز به سیستم جستجوی قوی در مدیریت
*   امکان نمایش فاکتور بعد از پرداخت موفق به کاربر جهت پرینت
*   کاملا ریسپناسیو و با هر قالبی میتونه کار کنه
*    تغییر رنگ فرم توسط مدیریت جهت مچ شدن با قالب سایت
*   و کلی از امکانات قوی که در موارد مشابه پیدا نمیشه.
*   قابلیت ریدایرکت کاربر بعد از پرداخت به صفحه دلخواه

قابلیت ویژه:

+	دادن لینک مقدار دار به کاربر (با استفاده از این قابلیت وقتی کاربر لینک را باز می کند با فرمی مواجه می شود که مبلغ آن از پیش با مقداری که درون لینک است ست می شود.)
